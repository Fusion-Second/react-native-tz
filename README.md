# react-native-tz

Get users TimeZone based on location.

## Installation

```sh
npm install react-native-tz
```
for iOS:

```sh
cd ios && pod install
```
## Usage

```js
import { getTimeZone } from "react-native-tz";

// ...

const result = await getTimeZone();
```
